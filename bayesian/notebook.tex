\emph{Poisson}-distributed

\[P(Z = k) =\frac{ \lambda^k e^{-\lambda} }{k!}, \; \; k=0,1,2, \dots \]

If a random variable \(Z\) has a Poisson mass distribution, we denote
this by writing

\[Z \sim \text{Poisson}(\lambda) \]

With tenserflow probability module we can easily apply mcmc algorithm to
modify our distribuiton and generate the best result
